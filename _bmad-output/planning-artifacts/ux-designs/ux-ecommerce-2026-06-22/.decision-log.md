## Major Strategic Decision

### D1: Brand Pivot — From General E-Commerce to Oren Premium Luxury
**Decision**: Redesign entire UX system from warm boutique commerce to **Oren: premium women's fashion luxury brand**. 

**Shift**: 
- **From**: General warm boutique commerce (business-neutral, Nunito + system fonts, beige + indigo)
- **To**: Premium women's fashion editorial brand (Oren, Nunito exclusive, soft pastel palette, emotional + aesthetic-driven)

**Rationale**: Positioning as luxury fashion requires:
- Feminine, soft, calm aesthetic (not neutral/corporate)
- Editorial/storytelling-driven discovery (not algorithmic search-first)
- Whitespace and silence as design elements (luxury hallmark)
- Minimal information density (emotion over specs)
- Exclusive typography (Nunito only)

**Implications**:
- **Typography**: Nunito exclusively (no system fallbacks) — strict consistency, luxury marker
- **Color palette**: Warm beige + pastel only (remove tech-style indigo completely)
  - Warm beige `#f5f1ed` (primary background)
  - Ivory `#faf8f5` (secondary background)
  - Sand beige `#e8d7c6` (cards/surfaces)
  - Muted blush pink `#e8cfc4` (subtle accent)
  - Soft clay `#d4a89a` (hover states/highlights)
- **Component design**: Minimal, elegant, CTA-on-hover-only (not always visible)
- **Product cards**: Image-dominant, name + price only, remove clutter
- **PDP**: Editorial layout, large image gallery, "Styling Notes" as magazine copy
- **Tone**: Fashion magazine + luxury boutique showroom (not e-commerce dashboard)

---

### D2: Form Factor Strategy (Unchanged)
**Decision**: Mobile-first shopper (70% focus, fully responsive), desktop-first admin (100% optimized).

**Rationale**: Same as v1 — mobile is discovery device, desktop for cart/checkout.

---

### D3: Critical Loop Focus (Refined for Fashion)
**Decision**: Prioritize Discovery → PDP → Add to Cart loop with **editorial/emotional tone** instead of speed/efficiency.

**Rationale**: Fashion discovery is about inspiration and aspiration, not transactional speed. User spends time with product (reads styling notes, imagines wearing), then converts from desire not urgency.

**Implications**:
- Discovery: Editorial storytelling (collections, mood boards) alongside search
- PDP: Large image gallery, styling narrative, minimal specs
- Add-to-cart: Subtle, integrated, not aggressive CTA
- Timing: No rushing user (slow, thoughtful experience)
